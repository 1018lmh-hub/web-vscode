2. 자식선택자와 후손선택자

자식 : 바로 하위 요소
후손 : 하위 요소들 전부 

        -자식선택자
        a > b {
            스타일 속성 : 값;
            ...
        }
        -후손선택자
        a b{
            스타일속성 : 값;
            ...
        }

3. 동위(같은 레벨) 선택자

        -a요소 뒤에 있는 b요소 딱 하나만을 선택
        a + b{
            스타일 속성 : 값;
            ...
        }

        -a요소 뒤에 있는 모든 b요소를 선택
        a ~ b{
            스타일 속성 : 값;
            ...
        }

4.반응 선택자

        -해당 요소를 클릭하고 있을 때
        선택자: active{
            스타일 속성 : 값;
        }

        -해당 요소(border)에 마우스커서가 올라가는 순간 선택자
        선택자:hover{
            스타일속성: 값;
        }
