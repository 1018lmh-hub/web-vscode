아직도 너무 많이 남아있는 jQuery란 무엇인가?

궁금쓰

기존 자바스크립트만으로는 (옛날얘기) 브라우저마다 각기 다른 코드를 작성해서
브라우저를 조작해야하는 어려움이 있었다!!
창시자 "존 레식"은 이런 문제를 해결하기 위해 JavaScript라는 언어를 이용해서
함수모음집을 만들어서 배포했다!!
이것이 바로 jQuery 라는 라이브러리라는 것이다!
javascript로 함수 많이 모아놓은 파일 == 라이브러리
- 사람들을 대체 왜 그렇게 jQuery에 열광했나요?
1) DOM요소와 관련된 스크립트 코드를 아주 쉽게 구현할 수 있음
2) AJAX (비동기식 통신), 이벤트 처리를 아주 쉽게 구현할 수 있음
3) jQuery와 관련된 확장 플러그인, Open API등이 많이 지원된다.
=> 조금 쓰고 많이 할 수 있음! 단점 => 바닐라 자바스크립트 대비 속도가 너무 느림, 그래서 요즘 신규개발엔 사용되지 않음
