자바스크립트

웹상에서 가장 많이 사용하는 스크립트 언어 현시대 웹에서 화면을 만들어내는 기반 기술로 사용되는 것이 가장 큰 특징 (Angular, Vue, React) 서비스 기업(네이버나 카카오)는 Vue를 많이 SI기업은 React를 많이 씀
- 컴파일 : 컴파일러가 컴파일 단계를 거쳐서 실행
-인터프리터 : 필요할 때 마다 한줄 한줄 바로바로 실행
JIT 컴파일러 => 자주 실행되는 "핫 코드"를 기계어로 재 컴파일(컴파일 인터프리터 둘 다 해당, 바이트 코드를 기계어로 컴파일)

자바 / 자바스크립트 => 컴파일 + 인터프리터

  • 함수를 일급 객체로 취급함(JS는 인자값이 아니라 함수 자체를 전달 가능)
  • 문법이 쉬움
    접근제한자 개념이 X, 함수 반환형 X , 변수 타입 지정 X(타입추론)
  • 계속 발전하고 피드백이 빠름 확장성이 좋아 배워두면 여기저기 사용 가능

자바스크립트 소스코드 작성 위치에 따른 종류

  1. inline : 요소 속성에 간단한 자바스크립트 코드를 작성
  2. internal : HTML 문서 내에 script 태그에 코드를 작성
  3. external : 코드를 별도의 .js 파일로 작성해서 읽어오는 방법 (모듈로 만들어서 export하고 쓰는 쪽에서 import해서 불러오는 방법도 있음)
