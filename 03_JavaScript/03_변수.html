자바스크립트의 경우 자료형을 선언하지 않음

자바 스크립트는 총 세개의 키워드로 변수를 선언할 ㅅ ㅜ있음

*var,let,const

변수 선언 위치에 따른 전역 / 지역변수

* 자료형

    변수 선언 시 자료형을 별도로 선언하지 않음
    자료형의 개념이 없는 건 아님=
    => 변수에 대입되는 값에 따라서 알아서 자료형을 결정함 : 타입 추론

자료형의 종류

  • string(문자열)
  • number(숫자)
  • boolean(논리)
  • object(객체)
  • function(함수)
  • undefined(초기화가 안된 상태)
